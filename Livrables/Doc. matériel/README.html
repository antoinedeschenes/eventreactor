Le code pour la lecture des thermomètres est basé légèrement sur celui fourni par Adafruit pour le capteur MCP9808, puis modifié pour le TMP007 : <br> MCP9808 Temperature Sensor Library

Fiches techniques dans le dossier :

Semiconducteurs:
  • STMicro ULN2003A : Sert à alimenter le relais inversant la phase de la plaque thermoélectrique;
  • CUI CP85438 : Plaque thermoélectrique;
  • Microchip MCP9808 : Thermomètre à contact direct;
  • Texas Instruments TMP007 : Thermomètre infrarouge.
Bloc d'alimentation :
  • B&K 1687B : Protocole de communication par port série du bloc d'alimentation à partir de la page 26.
